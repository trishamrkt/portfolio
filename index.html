rwalgebra.js

A math library for parsing, storing, and manipulating algebraic expressions. Supports operations in the real and complex domains.
—
Research regarding various math expression parsing techniques used in compilers was completed. Notably, the Shunting Yard, and post-fix notation were used in implementation.

Notable Features:
  • Arithmetic operations including: addition, subtraction, multiplication, division, inversion
  • Algebraic expression string parsing
    • Expression evaluation
  • Provide user access to each term in an expression
  • Display expressions in polar or cartesian coordinates
Github Link: https://github.com/capstone2019-2020/rwalgebrajs
Demo: https://trishamrkt.github.io/rwalgebra/
Tool(s): NodeJS

analog circuit analysis tool

Analog circuit analysis tool for signal flow graph generation, and loop gain & transfer function calculation.
—
A full-year capstone design project aimed to aid circuit designers in analog circuit stability analysis. Worked closely with a professor at the University of Toronto to generate a detailed design doc and ensure project requirements were satisfied.

Features Implemented:
  • Mason's rule algorithm
  • Dashboard UI
  • Node-edge graph visualization using Cytoscape.js library
  • Graph export into PNG and JPG format
  • Circuit schematic visualization using SVG elements
  • Unit tests using the MochaJS framework
Github Link: https://github.com/capstone2019-2020/Capstone
Tool(s): NodeJS, HTML, CSS, Javascript

booklr

Book suggestion webpage.
—
A user is able to search for a book and the application will provide a list of books that are similar to the entered book. The application uses Python to perform web-scraping in order to collect book data. The logic behind the suggestions relies mainly on the genre of the book and the author.

Notable Features
  • Search bar with automatic suggestions
  • Links to purchase the book on Amazon or Chapters
  • User can like/favourite books and view them later
Github Link: https://github.com/trishamrkt/book-suggestion-page
Demo: https://trishamrkt.github.io/book-suggestion-page/
Tool(s): Python, Javascript, AngularJS, MySQL
